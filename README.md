# WhisperNet API

Backend API for WhisperNet, a social platform with content moderation and user management.

## Features

- Content moderation using OpenAI
- Firebase authentication
- Post management with approval system
- Comment system
- User ranking and leaderboard
- Search functionality
- Incognito posting mode

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file with required environment variables
3. Start the server:
```bash
npm start
```

## Environment Variables

Create a `.env` file with:
- PORT
- OPENAI_API_KEY
- Firebase Admin SDK credentials
